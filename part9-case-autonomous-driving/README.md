# 第九章 · 案例研究：自动驾驶数据平台后端架构

> 一个真实的岗位 JD 拆解，把全书知识体系对准一个具体领域——自动驾驶数据平台。
> 看看"全栈开发"在自动驾驶行业的数据闭环场景下，需要哪些知识，哪些我们已覆盖，哪些需要补。

## 本章定位

这不是教学章，而是**知识体系的应用与验证**。拿一个真实的 Shopee/大疆/百度 Apollo 岗位 JD，逐条拆解技术要求，映射到全书的章节内容上，找到知识缺口，并给出面试准备路径。

这个案例研究的方法论可以复用到任何技术岗位——拆 JD → 映射知识体系 → 识别缺口 → 制定补缺计划。
